A personal couple website built with Next.js, React, Tailwind CSS, Framer Motion, and a backend API route.

## Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the site.

### What’s included

- Animated page transitions and reveal effects using `framer-motion`
- Responsive Tailwind CSS styling with light/dark theme toggle
- Interactive love note generator
- Photo timeline and polaroid gallery with flip animations
- Live countdown to your next anniversary milestone
- Backend API route at `/api/couple`

### Customize

Edit `src/app/page.tsx` and the component files in `src/components/` to personalize names, photos, captions, and dates.
